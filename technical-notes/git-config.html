Git Config

Configure identity, defaults, and aliases once to avoid repeated mistakes.

Must-Have Setup

Global configuration

PurposeCommand
Namegit config --global user.name "Your Name"
Emailgit config --global user.email "you@example.com"
Default branchgit config --global init.defaultBranch main
Rebase on pullgit config --global pull.rebase true

Useful Aliases

Faster workflow

  • git config --global alias.st status
  • git config --global alias.lg "log --oneline --graph --decorate"
  • git config --global alias.co switch
  • git config --global alias.br branch
