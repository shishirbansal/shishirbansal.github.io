Git Branch

Use branches to isolate work and keep main stable.

Branch Commands

Create, list, switch, rename, delete

ActionCommand
List local branchesgit branch
List all branchesgit branch -a
Create branchgit branch feature/auth
Create and switchgit switch -c feature/auth
Rename current branchgit branch -m new-name
Delete merged branchgit branch -d feature/auth

Best Practices

Keep history easy to review

  • Use short-lived feature branches.
  • Rebase frequently on latest main.
  • Name branches by change intent, not ticket-only.
