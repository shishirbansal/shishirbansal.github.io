Merge Commands

Most-used patterns

ActionCommand
Merge feature into maingit merge feature/auth
Allow merge commitgit merge --no-ff feature/auth
Fast-forward onlygit merge --ff-only feature/auth
Abort ongoing mergegit merge --abort

Conflict Flow

Repeatable steps

  • Run git status to identify conflicts.
  • Edit conflicted files and keep final expected code.
  • Stage resolved files with git add.
  • Complete with git commit.
