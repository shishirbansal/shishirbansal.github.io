Git

Introduction to Git

Git tracks snapshots of your code so you can collaborate, review changes, and recover safely.

Core Concepts

Working tree, staging, commits, remote

  • Working directory: your current files.
  • Staging area: selected changes for next commit.
  • Commit: saved snapshot with message.
  • Remote: shared repository on GitHub.
